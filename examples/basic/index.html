Fancy style guide

Buttons

Add a class of .button to a <button> or <a>element, and use a size modifier (small, large, etc) and an appearance modifier (primary, success).

Normal button

Use the .button class.

Confirm Success 
<a class="button">Confirm</a>  <a class="button primary">Success</a>

Button sizes

Use the .button class.

Confirm Success 
<a class="button small">Confirm</a>  <a class="button large">Success</a>
